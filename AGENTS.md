# BAO — Agent Instructions

## page-builder

Construir o reconstruir páginas fieles al diseño. Usar siempre el design system y las medidas documentadas.

### Medidas obligatorias (secciones tipo Wellness)

- **Título principal**: 44px, color `#2E394B`, font-weight 400
- **Subtítulo**: 25px, color `#D5AD7F`
- **Títulos de bloque** (ej. CONTENT, THE IMPACT): 25px, color `#2E394B`
- **Iconos de sección**: 37.5px × 37.5px

Consultar `.cursor/rules/bao-design-measures.mdc` y `CLAUDE.md` para más detalles.
